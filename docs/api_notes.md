# 飞书开发API重要特性记录

## 文档块插入特性

### CreateDocumentBlockDescendant API 关键特性

当使用 `CreateDocumentBlockDescendantRequest` 接口时：

#### 1. 根级别插入特性
```python
.document_id("V5HQdxo30ocXwNxT20KcJ20jn6b")
.block_id("V5HQdxo30ocXwNxT20KcJ20jn6b")  # block_id = document_id
```

**重要发现**：
- 当 `block_id = document_id` 时，新插入的内容会**添加到文档的开头**
- **不会覆盖**原有的文档内容
- 原有内容会**向下移动**
- 每次插入都是**追加操作**，不是替换操作

#### 2. 插入位置控制
```python
.index(0)  # 指定插入位置为0（开头）
```

#### 3. 实际测试结果
- 第一次运行：插入标题+表格，文档版本变为3
- 第二次运行：再次插入标题+表格，文档版本变为14
- 结果：文档开头现在有两个相同的"简单表格"标题和表格

#### 4. 应用场景
这个特性非常适合：
- **更新已有文档**：在文档开头添加新的更新内容
- **版本记录**：在文档顶部添加版本说明或更新日志
- **Markdown转换**：将转换后的内容插入到已有文档中
- **报告生成**：在现有报告文档中添加新的章节

#### 5. 注意事项
- 文档版本号会递增
- 每次操作都会在开头添加内容，而不是替换
- 需要合理规划内容结构，避免重复插入相同内容
